# Genshin-Artifacts-API
Uma API que exibe todos os conjuntos de artefato do jogo Genshin Impact.

# Genshin Artefatos

Projeto desenvolvido para estudar Python, Streamlit e consumo de APIs.

## Tecnologias
- Python
- Requests
- Streamlit

## Funcionalidades
- Consulta a API do Genshin Impact
- Exibe a lista de artefatos disponíveis

## API utilizada
